Início dessa jornada

Bem-vindo ao meu portfólio! Sou o Pedro, um profissional determinado e entusiasmado que está em meio a uma emocionante transição de carreira para o mundo do desenvolvimento full stack. Atualmente, estou aprimorando minhas habilidades por meio de um curso intensivo na EBAC, onde mergulhei profundamente em diversas tecnologias fundamentais.

Experiência Atual

Minha jornada começou com uma sólida base em tecnologias web essenciais. Até agora, dominei HTML, CSS e JavaScript, construindo a estrutura visual e a lógica por trás de páginas web envolventes e funcionais.

Outras tecnologias estudadas

JavaScript e jQuery: Aprofundei-me no JavaScript, explorando suas potentes capacidades para tornar as interações do usuário mais dinâmicas e intuitivas. Além disso, adquiri proficiência no uso do jQuery para simplificar e agilizar o desenvolvimento, trazendo eficiência aos meus projetos. Estudei Gulp e o GIT

Próximos passos

Estou entusiasmado com a perspectiva de expandir ainda mais minhas habilidades. No próximo estágio do curso, estarei explorando mais a fundo o desenvolvimento full stack, incluindo tecnologias como Python e outros conceitos essenciais para a construção de aplicativos web completos.
